隱私權政策

最後更新日期：2026年5月17日

歡迎使用 Daily Diet。我們非常重視您的隱私，本應用程式採用無帳號設計，讓您掌控自己的資料。

1. 資料收集

本應用程式不收集任何個人身分資訊，不需要登入帳號。所有飲食紀錄、目標設定及個人資料均儲存在您裝置的本地瀏覽器（IndexedDB）中。

2. 雲端備份（可選）

若您選擇啟用雲端備份功能，您的飲食紀錄將以 JSON 格式儲存至 GitHub Gist（私密模式）。私密 Gist 不會出現在任何搜尋結果中，僅能透過您的專屬 Gist ID 存取。我們不會存取、讀取或共享您的備份內容。

您隨時可以在設定介面刪除 Gist ID 或停止備份。

3. AI 辨識服務

當您使用 AI 食物辨識功能時，您拍攝的食物圖片（以 Base64 格式）將傳輸至 SiliconFlow API 進行分析。我們不會儲存您傳輸的任何圖片或分析結果。

4. 第三方服務

  • SiliconFlow：AI 食物辨識（圖片在分析後即捨棄）
  • GitHub Gist：可選的私密雲端備份儲存
  • Nominatim / OpenStreetMap：可選的用餐地點自動填入

本應用程式不使用 Google 帳號、Google Drive 或任何廣告追蹤服務。

5. 聯絡我們

如果您有任何問題，請透過應用程式內的「意見回饋」功能與我們聯繫。
